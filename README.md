# WeatherDashboard

A weather dashboard app built with Python and Streamlit to practice Python development and related skills.

## Features
- Current weather conditions for any city worldwide
- 24-hour hourly forecast
- 7-day weekly forecast with temperature and humidity charts
- Interactive map showing the searched location
- Support for Fahrenheit and Celsius

## Tech Stack
- Python, Streamlit, Open-Meteo API, Chart.js, Leaflet

## Running Locally
pip install -r requirements.txt
streamlit run app.py

## About This Project
This project is created for learning and practicing Python programming and web development skills.
